# Task 3 - Data Visualization & Dashboarding

## 📌 Objective

The objective of this task is to create professional visualizations and an interactive dashboard using Python and Power BI. The project focuses on transforming raw business data into meaningful insights through charts, KPIs, and dashboards.

---

## 📂 Dataset

**Dataset Used:** Sample Superstore Dataset

The dataset contains information about:

- Orders
- Customers
- Products
- Categories
- Sales
- Profit
- Discount
- Quantity
- Regions
